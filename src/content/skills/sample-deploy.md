## 概要

プロジェクトのビルドとデプロイを自動化するスキルのサンプル。

## 使い方

Claude Code で以下のトリガーワードを入力:
- 「デプロイして」
- 「pushして」

## 手順

1. ビルドを実行: `pnpm build`
2. デプロイ先を確認
3. ユーザーの承認を得てデプロイ実行

> このファイルはサンプルです。`pnpm sync` で `~/.claude/skills/` の実際のスキルに置き換わります。
